# comparison 조합 가이드

## 기본 구성
hook → comparison ×2~3 → brand-closer

## 변형 구성
- hook → comparison → chart → comparison → brand-closer (데이터 비교)
- hook → comparison ×2 → quote → brand-closer (인용구 마무리)

## 레이아웃 조합 규칙
- 첫 슬라이드는 반드시 hook
- 마지막 슬라이드는 반드시 brand-closer (전체 다크 배경, 상단 인용문 + 하단 브랜드)
- comparison 슬라이드 간에 다른 레이아웃을 끼워넣어 단조로움 방지
- 총 슬라이드 수: 4~7장 권장
- hook은 hook(단색) 또는 hook-photo(사진 배경) 중 선택 — 피드 노출 고려 시 hook-photo 권장
- brand-closer CTA 고정: "팔로우하고 다양한 콘텐츠를 경험해보세요"
